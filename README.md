## Quartz

The all-in-one extension that sanitizes your data and enhances your prompts for better, safer AI converations. Quartz has two powerful features: Security, & Efficiency.
This lets every one of you never leak sensitive data again, and get better AI results instantly.

## How to test it yourself:

1. Clone the github repository using "https://github.com/kee5625/Prompt-protekt.git".
2. Open chrome://extensions.
3. Click on "Developer Mode"
4. Click on "Load Unpacked"
5. Select the folder where you have cloned this repository.
6. Go to your chatgpt or claude to test it out!
